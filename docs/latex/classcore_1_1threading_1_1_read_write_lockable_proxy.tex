\subsection{Detailed Description}
Acts as a \hyperlink{classcore_1_1threading_1_1_read_write_lockable}{Read\-Write\-Lockable} by proxy reference. Any \hyperlink{classcore_1_1threading_1_1_read_write_lockable}{Read\-Write\-Lockable} calls get forwarded to the proxy instead, so the \hyperlink{classcore_1_1threading_1_1_read_write_lockable_proxy}{Read\-Write\-Lockable\-Proxy} appears as a \hyperlink{classcore_1_1threading_1_1_read_write_lockable}{Read\-Write\-Lockable} without needing to implement it's locking protocol.
